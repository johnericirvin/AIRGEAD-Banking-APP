# AIRGEAD-Banking-APP
I have created a GitHub repository containing my Airgead Banking Investment Calculator project, which demonstrates my ability to develop a structured and well-documented C++ application. This program allows users to input financial data and generates reports comparing investment growth with and without monthly deposits using compound interest calculations. Throughout this project, I focused on writing clean, readable, and maintainable code by applying object-oriented programming principles and organizing functionality into clearly defined methods. I also implemented strong input validation to ensure accurate and user-friendly interactions. While the program meets all functional requirements, future improvements could include adding file output, a graphical user interface, and enhanced modularity. This project strengthened my problem-solving, debugging, and programming skills, all of which are transferable to future coursework and professional development.
